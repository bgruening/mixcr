Usage: mixcr mergeAlignments [--force-overwrite] [--no-warnings] [--verbose] [--help] input.
                             vdjca... output.vdjca
Merge several *.vdjca files with alignments into a single alignments file.
      input.vdjca...       Paths to input files with alignments
      output.vdjca         Path where to write merged alignments
  -f, --force-overwrite    Force overwrite of output file(s).
      -nw, --no-warnings   Suppress all warning messages.
      --verbose            Verbose messages.
  -h, --help               Show this help message and exit.
